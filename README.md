# MgekX
第一个uni-app开发的Mgek集合安卓程序，集成了博客，Mgek项目，mgek云和小火箭爬虫。支持邮件订阅和日志查询

## 平台

Android 4.0+

## 需求

uni-ui插件

## 下载

[mgekx](https://github.com/Landers1037/mgekx/releases/tag/1.0.1)
